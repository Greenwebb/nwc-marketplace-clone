# Motta Product Detail Page - Complete Layout Analysis

## Page Structure (from screenshots)

### Top Section
1. **Breadcrumb** - Home > Category > Subcategory
2. **Social Share Icons** - Facebook, Twitter, Google+, Pinterest (top right)
3. **Product Title** - Large heading with category badge above
4. **Rating & Review** - Stars, review count, "Write a Review" link

### Main Content - Two Column Layout

#### Left Column - Image Gallery
- Large main product image (white background)
- Vertical thumbnail strip on left (4 thumbnails)
- Zoom icon indicator
- Image counter "1/4, 2/4, 3/4, 4/4"

#### Right Column - Product Info
- **Price Section**
  - Large current price ($233.98)
  - Strikethrough original price ($267.00)
  - Red savings badge "Save: $34.00 (12%)"
  
- **Stock Status** - Green text "Available in stock"

- **Quantity Selector** - Input with +/- buttons

- **Action Buttons**
  - "Add to cart" - Large blue button
  - "Buy Now" - Large outlined button
  - Full width on mobile, side-by-side on desktop

- **Secondary Actions**
  - "Compare" link with icon
  - "Add to Wishlist" link with icon
  
- **Store Info Card**
  - Store name with logo
  - Store rating
  
- **Ask a Question** - Dark button

### Tabs Section (Full Width)
- Tab headers: Description, Reviews (1), More Offers, Store Policies, Inquiries
- Content area below with collapsible sections

### Similar Products Section
- Heading "Similar Products"
- Horizontal carousel with product cards
- Each card shows: image, title, rating, price, store name
- Navigation arrows for scrolling

### Benefits Bar (Bottom)
- 4 benefit cards in a row
- Icons + text for: Worldwide Delivery, Secure Payment, 60-day Return Policy, 24/7 Help Center

## Design Specifications
- Container max-width with padding
- White background for main content
- Light gray background for page
- Blue primary buttons (#11248F)
- Red/coral for savings and sale prices
- Green for stock status
- Clean spacing and typography
